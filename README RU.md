# Krauser Discord Cloner

Современное приложение для клонирования и резервного копирования Discord-серверов.
Тёмный интерфейс, безопасное хранение токена, авто-обновление.

![version](https://img.shields.io/badge/version-3.1-blue)
![platform](https://img.shields.io/badge/platform-Windows-lightgrey)

---

## Возможности

- **Клонирование** — полная копия сервера (роли, каналы, эмодзи, стикеры, вебхуки и др.)
- **Синхронизация** — добавить только недостающие объекты, ничего не удаляя
- **Очистка** — полное удаление содержимого сервера
- **Бэкап и восстановление** — сохранение структуры сервера в файл
- **Авто-бэкап** перед удалением — страховка ваших данных
- **Профили настроек** — сохранённые наборы параметров
- **История операций** — журнал всех действий
- **9 цветовых тем**, светлая/тёмная тема, русский и английский языки
- **Авто-обновление** через GitHub

## Установка

1. Скачайте **`KrauserCloner-Setup.exe`** из раздела [Releases](https://github.com/james4536487-dot/Krauser-Discord-Cloner/releases/tag/v3.1)
2. Запустите установщик и следуйте мастеру установки
3. Запустите приложение с рабочего стола или из меню Пуск

Или используйте портативную версию **`KrauserCloner.exe`** — без установки.

---

## Безопасность

- Токен шифруется через **Windows DPAPI** и привязан к вашей учётной записи —
  даже при копировании файла токен нельзя расшифровать на другом ПК.
- Перед удалением данных автоматически создаётся резервная копия.

## Внимание

Эта программа работает с вашим личным токеном Discord, то есть, по сути, как селф-бот. 
Discord это запрещает, и из-за этого могут заблокировать ваш аккаунт. 
Я сразу говорю, что не отвечаю за то, что ваш аккаунт могут заблокировать. 
Вы сами решаете скачать этот файл и запустить клонирование. 
Если вы это делаете, значит, вы соглашаетесь, что я не виноват, если с вашим аккаунтом что-то случится. 
Если вы совсем не хотите, чтобы вас забанили, то лучше используйте для этого какой-нибудь запасной аккаунт, который не жалко. 
А потом уже передадите права сервера на основной аккаунт.
